# ura_study_hub
Justificarea Stack-ului Tehnologic (Milestone 1)
Pentru dezvoltarea aplicației, am ales un set de tehnologii moderne (MERN Stack adaptat) care să asigure scalabilitatea, securitatea și rapiditatea dezvoltării.  
• Frontend: React.js – Ales pentru capacitatea de a crea o interfață de utilizator (UI) dinamică și rapidă, esențială pentru a menține angajamentul celor 20+ utilizatori live.  
• Backend: Node.js cu Express – Această tehnologie permite gestionarea eficientă a cererilor asincrone și este ideală pentru funcționalități de tip chat sau încărcare de fișiere în timp real.  
• Bază de Date: MongoDB – Oferă o structură flexibilă pentru stocarea diverselor tipuri de materiale de studiu (PDF, imagini, text) identificate ca necesare în urma cercetării.  
• Hosting & Deployment: Vercel / Render – Platforme care asigură o conexiune continuă cu repository-ul de GitHub și oferă un link public stabil pentru evaluatori.  
• Analytics: Google Analytics / Mixpanel – Esențiale pentru Milestone 3, aceste unelte vor monitoriza retenția săptămânală a utilizatorilor, conform cerințelor de evaluare.  
Metodologia de Dezvoltare (AI-Assisted Coding)
Deoarece dezvolt proiectul individual, voi utiliza un flux de lucru asistat de Inteligență Artificială (AI-assisted coding) pentru a accelera implementarea.  
• Workflow: Voi folosi AI pentru generarea structurilor de bază (boilerplate), depanarea erorilor (debugging) și optimizarea interogărilor bazei de date.  
• Documentare: Toate prompt-urile utilizate vor fi salvate și citate în varianta finală a proiectului, asigurând transparența procesului de dezvoltare.
